# Text Processing using Python (Regex & Data Analysis)

## 📌 Project Overview
This project demonstrates **text processing and analysis using Python**. It uses **Regular Expressions (Regex)** to extract useful information from a sample text and performs basic text analytics with visualizations using **Pandas** and **Matplotlib**.

## 🚀 Features

- Extract Email Addresses
- Extract Phone Numbers
- Extract Dates
- Extract URLs
- Extract Prices
- Extract Time Values
- Extract Hashtags
- Find Words Ending with "ing" and "tion"
- Word Frequency Analysis
- Stopword Removal
- Word Length Distribution
- Top 10 Longest Words
- Data Visualization using Matplotlib

## 🛠️ Technologies Used

- Python 3
- Regular Expressions (re)
- Pandas
- Matplotlib
- Collections (Counter)
